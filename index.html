8085 Microprocessor Overview

The Intel 8085 is an 8-bit microprocessor that was introduced by Intel in 1976. It is the last 8-bit microprocessor developed by Intel. The 8085 uses a single +5V power supply and operates at 3 MHz, 5 MHz, and 6 MHz versions.

Key Features

  • 8-bit data bus
  • 16-bit address bus
  • 64KB addressable memory
  • 6 general-purpose registers
  • 5 interrupt signals
  • Serial I/O ports
  • DMA capability

Technical Specifications

  • Clock Speed: 3-6 MHz
  • Instruction Set: 246 instructions
  • Addressing Modes: 5
  • Power Supply: +5V
  • Technology: NMOS
  • Package: 40-pin DIP

Arithmetic Logic Unit (ALU)

Performs arithmetic and logical operations on 8-bit data.

Registers

Six 8-bit general-purpose registers (B, C, D, E, H, L) that can be paired to form three 16-bit registers (BC, DE, HL).

Program Counter (PC)

16-bit register that holds the address of the next instruction to be executed.

Stack Pointer (SP)

16-bit register that points to the top of the stack in memory.

Arithmetic Logic Unit (ALU)

The Arithmetic Logic Unit (ALU) is the computational heart of the 8085 microprocessor. It performs arithmetic operations (addition, subtraction) and logical operations (AND, OR, XOR, etc.) on 8-bit data.

ALU Functions

  • Addition and subtraction
  • Logical AND, OR, XOR
  • Increment and decrement
  • Bit shifting and rotation
  • Compare operations
  • Decimal adjustment

ALU Operations

  • All arithmetic operations use the accumulator as one operand
  • Results of ALU operations are stored in the accumulator
  • ALU sets flags in the flag register based on operation results
  • Supports both 8-bit and 16-bit operations

Accumulator (A)

8-bit register used for arithmetic, logic, and data transfer operations. One operand is always stored in the accumulator.

Temporary Register

8-bit register used to hold temporary data during ALU operations. Not accessible to the programmer.

Flag Register

5-bit register that stores the status of arithmetic and logical operations (Sign, Zero, Auxiliary Carry, Parity, Carry).

ALU Control

Controls the operation of the ALU based on the instruction being executed.

Registers

The 8085 microprocessor contains several types of registers that store data, addresses, and status information during program execution.

General Purpose Registers

  • B, C, D, E, H, L - Six 8-bit registers
  • Can be used individually or in pairs (BC, DE, HL)
  • HL pair is often used as a memory pointer
  • Used for data manipulation and temporary storage

Special Purpose Registers

  • Accumulator (A) - Primary register for ALU operations
  • Program Counter (PC) - 16-bit address of next instruction
  • Stack Pointer (SP) - 16-bit address of stack top
  • Flag Register :-

Accumulator (A)

8-bit register used as the primary operand for arithmetic and logical operations. Results are stored here.

General Purpose Registers

B, C, D, E, H, L - Six 8-bit registers that can be used individually or in pairs (BC, DE, HL) for 16-bit operations.

Program Counter (PC)

16-bit register that holds the memory address of the next instruction to be executed.

Stack Pointer (SP)

16-bit register that points to the top of the stack in memory, used for subroutine calls and interrupts.

Flag Register

5-bit register that stores the status of arithmetic and logical operations (Sign, Zero, Auxiliary Carry, Parity, Carry).

Instruction Register (IR)

8-bit register that holds the opcode of the current instruction being executed.

Control Unit

The Control Unit coordinates and controls all operations of the microprocessor. It generates control signals that regulate the flow of data between the CPU, memory, and I/O devices.

Control Signals

  • ALE (Address Latch Enable): Used to latch the lower byte of the address from the multiplexed address/data bus.
  • RD (Read): Indicates the processor is reading data from memory or an I/O device.
  • WR (Write): Indicates the processor is writing data to memory or an I/O device.
  • IO/M (I/O or Memory): Distinguishes between I/O (1) and memory (0) operations.
  • S0, S1 (Status): Indicate the current operation or machine cycle status of the processor.
  • READY: Signals whether a peripheral is ready for data transfer or needs wait states.

Timing and Control

  • Instruction fetch cycle
  • Memory read/write cycle
  • I/O read/write cycle
  • Interrupt acknowledge cycle
  • Bus idle cycle

Instruction Decoder

Decodes the instruction fetched from memory and determines the operations to be performed.

Timing and Control Unit

Generates control signals and coordinates the timing of all microprocessor operations.

Machine Cycle Controller

Controls the sequence of operations during instruction execution (fetch, decode, execute).

Interrupt Controller

Manages the five hardware interrupts (TRAP, RST 7.5, RST 6.5, RST 5.5, INTR).

Bus Structure

The 8085 microprocessor uses three types of buses to communicate with memory and I/O devices: Address Bus, Data Bus, and Control Bus.

Address Bus

  • 16-bit unidirectional bus
  • Can address 64KB memory
  • A15-A8: Higher address lines
  • AD7-AD0: Multiplexed address/data lines

Data Bus

  • 8-bit bidirectional bus
  • Transfers data between CPU and peripherals
  • AD7-AD0: Multiplexed address/data lines

Address Bus

16-bit unidirectional bus used to specify memory locations or I/O ports. A15-A8 are dedicated address lines, while AD7-AD0 are multiplexed with data.

Data Bus

8-bit bidirectional bus used for data transfer between the microprocessor and memory or I/O devices.

Control Bus

Consists of various control signals (RD, WR, ALE, IO/M, etc.) that coordinate data transfer and operation sequencing.

Bus Demultiplexing

ALE signal is used to separate address and data on the multiplexed AD7-AD0 lines using an external latch.

Instruction Set

The 8085 has a rich instruction set with 246 instructions. These instructions can be classified into different categories based on their functionality.

Instruction Categories

  • Data Transfer Instructions
  • Arithmetic Instructions
  • Logical Instructions
  • Branch Instructions
  • Stack, I/O, and Machine Control

Instruction Formats

  • One-byte instructions
  • Two-byte instructions
  • Three-byte instructions

Data Transfer Instructions

MOV, MVI, LXI, LDA, STA, LHLD, SHLD, LDAX, STAX, XCHG, etc. Used to transfer data between registers and memory.

Arithmetic Instructions

ADD, ADI, ADC, ACI, SUB, SUI, SBB, SBI, INR, DCR, INX, DCX, DAD, DAA. Perform arithmetic operations on data.

Logical Instructions

ANA, ANI, ORA, ORI, XRA, XRI, CMP, CPI, RLC, RRC, RAL, RAR, CMA, CMC, STC. Perform logical operations on data.

Branch Instructions

JMP, JC, JNC, JZ, JNZ, JP, JM, JPE, JPO, CALL, RET, RST. Control program flow and subroutine calls.
